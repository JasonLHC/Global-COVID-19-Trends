# Global COVID-19 Trends Using the Johns Hopkins CSSE Dataset

This repository contains an R Markdown analysis of global COVID-19 trends using the Johns Hopkins CSSE dataset.

## Files
- `Global-COVID-19-Trends-Using-the-Johns-Hopkins-CSSE-Dataset.Rmd`

## Data Source
The report imports data directly from the Johns Hopkins CSSE GitHub repository using raw CSV links.

## Notes
The project is fully reproducible as long as the source URLs remain available and the required R packages are installed.
